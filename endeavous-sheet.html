Endeavour OS Terminal Cheat Sheet

Endeavour OS est une distribution basée sur Arch Linux, conçue pour être légère et facilement personnalisable. Elle utilise principalement un terminal pour la gestion des paquets, des mises à jour et l'exécution des tâches administratives. Endeavour OS offre aussi l'outil yay pour la gestion des paquets AUR (Arch User Repository), facilitant l'installation de logiciels supplémentaires.

Navigation dans le terminal

ls

Lister les fichiers dans le répertoire courant

cd [répertoire]

Changer de répertoire

pwd

Afficher le chemin du répertoire courant

mkdir [nom]

Créer un nouveau répertoire

Manipulation des fichiers

cat [fichier]

Afficher le contenu d'un fichier

grep [mot-clé] [fichier]

Rechercher une chaîne spécifique dans un fichier

touch [fichier]

Créer un nouveau fichier

rm [fichier]

Supprimer un fichier

cp [source] [destination]

Copier un fichier ou un dossier

mv [source] [destination]

Déplacer ou renommer un fichier ou dossier

Commandes système

sudo [commande]

Exécuter une commande en tant qu'administrateur Utilisez avec précaution!

df -h

Afficher l'utilisation du disque

top

Afficher les processus actifs en temps réel

ps aux

Lister les processus en cours d'exécution

kill [PID]

Terminer un processus spécifique

Commandes réseau

ping [adresse]

Envoyer des requêtes ping à une adresse IP ou un domaine

ifconfig

Afficher les informations réseau de votre machine

netstat -tuln

Afficher les connexions réseau actives

Gestion des paquets avec Pacman

sudo pacman -S [paquet]

Installer un paquet depuis les dépôts officiels

sudo pacman -R [paquet]

Désinstaller un paquet

sudo pacman -Sy

Synchroniser les dépôts sans installer

sudo pacman -Syu

Mettre à jour tout le système

Gestion des paquets AUR avec Yay

yay

Mettre à jour tous les paquets AUR et Arch

yay -S [paquet]

Installer un paquet depuis AUR

yay -R [paquet]

Désinstaller un paquet AUR

yay -Ss [nom_paquet]

Rechercher un paquet dans AUR

yay -Yc

Nettoyer les dépendances inutilisées

Autres commandes utiles

clear

Nettoyer le terminal

history

Afficher l'historique des commandes

alias [alias]='[commande]'

Créer un raccourci pour une commande

uname -r

Afficher la version du noyau Linux

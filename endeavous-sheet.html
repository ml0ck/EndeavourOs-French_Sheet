Endeavour OS Terminal Cheat Sheet

Endeavour OS est une distribution basée sur Arch Linux, conçue pour être légère et facilement personnalisable. Elle utilise principalement un terminal pour la gestion des paquets, des mises à jour et l'exécution des tâches administratives. Endeavour OS offre aussi l'outil yay pour la gestion des paquets AUR (Arch User Repository), facilitant l'installation de logiciels supplémentaires.

Navigation dans le terminal

ls

Lister les fichiers dans le répertoire courant

cd [répertoire]

Changer de répertoire

pwd

Afficher le chemin du répertoire courant

mkdir [nom]

Créer un nouveau répertoire

Suppression des traces et nettoyage des logs

history -c

Effacer l'historique des commandes de la session actuelle

rm ~/.bash_history

Supprimer le fichier d'historique des commandes pour l'utilisateur courant

sudo journalctl --vacuum-time=1d

Supprimer les logs du journal système datant de plus d'un jour

sudo rm -rf /var/log/*.log

Supprimer tous les fichiers de logs dans le répertoire /var/log

sudo truncate -s 0 /var/log/syslog

Vider le contenu du fichier syslog sans le supprimer

sudo rm -rf ~/.cache/*

Supprimer tous les fichiers de cache de l'utilisateur courant

sudo find / -name '*.log' -delete

Supprimer tous les fichiers de logs dans le système (utiliser avec précaution)

sudo rm -rf /tmp/*

Supprimer tous les fichiers temporaires dans le répertoire /tmp

Gestion des fichiers cachés

ls -a

Lister tous les fichiers, y compris les fichiers cachés dans le répertoire courant

ls -la

Lister tous les fichiers avec des détails, y compris les fichiers cachés

cd .[répertoire]

Accéder à un sous-répertoire caché

cp -r .[dossier] [destination]

Copier un dossier caché et son contenu vers un autre emplacement

mv .[fichier] [nouveau_nom]

Renommer ou déplacer un fichier caché

rm .[fichier]

Supprimer un fichier caché

touch .[fichier]

Créer un nouveau fichier caché

nano .[fichier]

Éditer un fichier caché avec l'éditeur de texte nano

Manipulation des fichiers

cat [fichier]

Afficher le contenu d'un fichier

grep [mot-clé] [fichier]

Rechercher une chaîne spécifique dans un fichier

touch [fichier]

Créer un nouveau fichier

rm [fichier]

Supprimer un fichier

cp [source] [destination]

Copier un fichier ou un dossier

mv [source] [destination]

Déplacer ou renommer un fichier ou dossier

Gestion des utilisateurs et accès root

sudo su

Passer en tant que root (superutilisateur)

su [utilisateur]

Changer d’utilisateur sans mot de passe (si déjà root) ou avec le mot de passe de l’utilisateur spécifié

sudo -i

Ouvrir un shell de connexion root avec les variables d’environnement de root

sudo passwd root

Changer ou définir le mot de passe root

Créer un nouvel utilisateur avec son répertoire personnel

sudo userdel [nom_utilisateur]

Supprimer un utilisateur

sudo usermod -aG [groupe] [utilisateur]

Ajouter un utilisateur à un groupe spécifique

groups [utilisateur]

Afficher les groupes auxquels appartient un utilisateur

sudo visudo

Modifier le fichier sudoers pour définir les permissions administratives des utilisateurs

sudo deluser [utilisateur] [groupe]

Supprimer un utilisateur d'un groupe

whoami

Afficher le nom de l'utilisateur courant

sudo [commande]

Exécuter une commande en tant qu'administrateur Utilisez avec précaution!

Afficher l'utilisation du disque

Afficher les processus actifs en temps réel

Lister les processus en cours d'exécution

Commandes réseau

ping [adresse]

Envoyer des requêtes ping à une adresse IP ou un domaine

Afficher les informations réseau de votre machine

Désinstaller un paquet

Gestion des paquets AUR avec Yay

gzip [fichier]

Compresser un fichier avec gzip

Compresser un fichier ou un répertoire en format zip

Sécurité et analyse des fichiers

sudo clamscan -r [chemin]

Mettre à jour la base de données des signatures de ClamAV

Autoriser le trafic sur un port spécifique via UFW

Vérifier l'état de Fail2ban pour bloquer les tentatives de connexion malveillantes

sudo netstat -tulnp
